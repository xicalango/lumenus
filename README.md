Lumenus
=======

A simple [shmup](http://en.wikipedia.org/wiki/Shmup) written in Lua for the [Love 2D engine](https://love2d.org/).

Requirements
============

- Love 2D (Tested with version 11.1)

Key bindings
============

- a, Left Arrow Key  
	Move Left
- d, Right Arrow Key  
	Move Right
- w, Up Arrow Key  
	Move Up
- s, Down Arrow Key  
	Move Down
- Space Bar  
	Shoot
- Shift  
	Modifier/"Focus"
- Enter  
	Select Entry
- Escape  
	Pause/Abort
- F5  
	Toggle fullscreen
- F9  
	Add 100000 credits
- F10  
	End level
- F12  
	Quit game
